**Directions: Extracting SVG Code and Importing into Figma**

1. **Access the SVG file**

   * Download the SVG or open it in your browser.

2. **Get the SVG code (choose one method)**

    **A. View the SVG code directly**

   * Right-click the SVG and choose **“View Page Source”** or **“Inspect.”**

   * Find the full `<svg>...</svg>` block.

   * Copy all of the SVG code.

3. **B. Convert the SVG file to view/copy its code**

   * Open the [SVG-to-code converter link](https://nikitahl.github.io/svg-2-code/).

   * Upload your SVG file.

   * Copy the generated code.

4. **Paste the SVG code into Figma (watch [video](https://www.youtube.com/watch?v=aKr1McI01uk&t=22s) for exact method)**

   * Open your Figma file.

   * Use the import method shown in the [video](https://www.youtube.com/watch?v=aKr1McI01uk&t=22s).

   * The SVG should appear as editable vector layers.

5. **Verify editability**

   * Select the imported graphic.

   * Make sure paths, shapes, and layers are individually editable.

   * **Note:** Text will show up as individual shapes, so you can change it to a text box either before pasting or after.
