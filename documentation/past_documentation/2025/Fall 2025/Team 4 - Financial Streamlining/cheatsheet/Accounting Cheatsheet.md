

| Terminology |  |
| ----- | :---- |
| Asset | Something you can invest in |
| Portfolio | Shows overall position across all investments |
| Profit / Gain | Earnings (Sell Price \- Buy Price) |
| Loss | Buy Price \- Sell Price  |
| Cost Basis | What you paid for the investment, used to calculate gains/losses for tax purposes |
| Capital Gain | Profit from selling an asset (taxable income) |
| Capital Loss | Loss from selling an asset (can lower your tax liability) |
| Realized | The asset has been sold & can affect your taxes |
| Unrealized | The asset has not been sold, so the event does not affect your taxes |
| Dividend | Money paid out to shareholders |
| Qualified Dividend | If holding period conditions are met, these distributions can be taxed at capital gains rates (favorable tax treatment) |
| Short \- Term Gain | Profit from asset held \< 1 year (higher tax rate) |
| Long \- Term Gain | Profit from asset held \> 1 year (lower tax rate) |
| Stock Split | When a company increases the number of shares resulting in a lower stock price which spurs investment (total value remains constant) |
| Recognized Gain | Portion of realized gain that IRS considers taxable in that year |

| Tax Forms |  |
| ----- | :---- |
| 1099-B | Reports sales of investments to calculate capital gains and losses |
| 1099-DIV | Reports dividends received when at least $10 in dividends have been received |
| 1099-INT | Reports interest earned |
| 1116 | Used to claim foreign tax credit, common with international dividends |
|  |  |

| Dividend Terminology |  |
| ----- | :---- |
| Dividend Payment | The actual money paid to the investor per share |
| Dividend Yield | A percentage showing how much a stock pays annually relative to its price |
| Ex-Dividend Date | To qualify to receive the dividend, you must own the stock before this date |
| Record Date | The company checks who is a shareholder on this date |
| Pay Date | The date the dividend is actually paid to your account |
| Declaration Date | When the company officially announces the dividend |
