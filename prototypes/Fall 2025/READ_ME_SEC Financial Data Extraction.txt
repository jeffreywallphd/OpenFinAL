How to run the script?

Navigate to the prototype folder \OpenFinAL\prototypes\Fall 2025 using command prompt or putty. Then run this script
python sec_snp500.py, the updated excel file of SEC_BS_IS_SP_500.xlsx will be recreated.

To add data into the table financial data in the database you run the following command node importFinancialData.js

Note: Make sure you have python installed in your machine
      If you get an error related to 'xlsx' use the command npm install xlsx
      If you get an error related to 'sqlite3' use the command npm install sqlite3
